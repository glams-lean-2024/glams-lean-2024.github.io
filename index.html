Lean learning group

This is the course webpage for the Lean learning group, a participant-led crash course in the Lean programming language, a functional programming language that can be used as a proof verification system. The course is running in Spring 2023 under the auspices of the Glasgow-Maxwell School. Organizational and logistical support is being delivered by David Jordan and Patrick Kinnear.
